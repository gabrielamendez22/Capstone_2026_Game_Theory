01 · Overview

What We Built

This report presents the first full cross-environment strategic profile estimation for six LLM model variants across three structurally distinct game-theoretic environments: Repeated Prisoner's Dilemma (PD), Commons Dilemma (CD), and Cheap-Talk Signaling (CT). The analysis operationalises the five-parameter strategic profile vector Sm = (ρ, θ, η, γ, β) defined in the project's conceptual framework and computes cross-environment coherence metrics as specified in Option 1.

80 PD raw files · 9,600 rounds
160 CD raw files · 12,843 rounds
112 CT raw files · 49,348 rounds

02 · Method

Parameter Estimation

All parameters are estimated at game level (one observation per complete game per model), then aggregated to model level. This avoids the within-game dependence inflation that round-level inference produces.

Note on θ: The original framework defines θ as the pool size at which extraction first exceeds the sustainable rate. In practice, all models over-extract from round 1, making the threshold degenerate (always = 100, the initial pool). We therefore operationalise θ as the normalised extraction ratio — a continuous measure of exploitation intensity — and recommend flagging this deviation in the Methods section.
Note on β (CD): Files generated under prompt v2.2 (April 2026) store beliefs in extraction units (0–20) rather than probabilities (0–1). The pipeline auto-detects this condition (max belief > 1.0) and normalises by dividing by 2× sustainable_share before computing MAE, ensuring cross-environment comparability.

Coherence Metrics

Following the Option 1 framework, we compute three coherence metrics at model level:

Cm = Var(βenv) — variance of mean β across the three environments. Lower = more stable belief calibration across games.

Cross-parameter correlations — Pearson r among ρ, θ, η, γ across models. High correlations would indicate a shared latent strategic disposition.

Human benchmark comparisons — Mann-Whitney U tests comparing each LLM parameter distribution against the relevant human prior (Dvorak & Fehrler 2024 for ρ; Abatayo & Lynham 2022 for θ; Gneezy 2005 for η and γ).

06 · Structural Coherence

Cross-Parameter Correlations

Interpretation: The η–γ correlation (r = 0.66, p = 0.154) is the strongest pairwise relationship. Models that send honest signals tend also to be more influenced by signals they receive — suggesting a coherent "trust orientation" that spans both CT roles. However, with n = 6 models this does not reach significance and should be interpreted cautiously. The absence of significant correlations among the other parameters supports the view that the profile vector captures structurally independent dimensions of strategic behaviour.
07 · Notable Patterns

Model-Level Observations

GPT-4o
Most coherent among frontier models (β var = 0.012). Reciprocity (ρ = 0.450) indistinguishable from human prior. Lowest exploitation in its family (θ = 4.17). Behaviorally closest to human benchmarks on ρ.
Gemini 2.5 Flash
Lowest reciprocity of all models (ρ = 0.009) — near-zero conditional cooperation, statistically indistinguishable from unconditional play. Highest exploitation (θ = 5.23). Yet belief calibration in CT is the best (βCT = 0.283).
Gemini 2.5 Flash Lite
Uniquely high γ = 0.607 — by far the most gullible receiver. Combined with η = 0.995 (near-perfect honesty), this model exhibits an extreme "trusting" profile. Most coherent by β variance but poorest βPD (0.495).
GPT-4o-mini
η = 0.998 — essentially never deceives as sender, but γ ≈ 0 — entirely unresponsive to sender messages as receiver. Strong asymmetry between sender and receiver roles.
Claude Opus
Largest model in the sample, least coherent (β var = 0.054). Human-like ρ (0.412, p = 0.998 vs. human), but βCD = 0.612 — worst belief calibration in the Commons environment by a large margin.
Claude Sonnet
Significantly less reciprocal than humans (ρ = 0.091, p < 0.0001). Highest exploitation among Claude family (θ = 4.17, but also βCD = 0.598). Overall profile closest to Claude Opus despite being a smaller model.
08 · Hypotheses

Preliminary Evidence on H1, H2, H3

H1 — Scaling increases coherence

Not supported   Claude Opus (largest) is the least coherent model (β var = 0.054). GPT-4o and Gemini Flash Lite show higher coherence. Within families, scaling does not monotonically improve cross-environment stability. This is one of the most striking findings and merits dedicated analysis.

H2 — Instruction tuning shifts the profile

Partially assessable   All models in the current dataset are instruction-tuned. Direct base vs. instruction-tuned comparisons are not available in this run configuration. Recommending this as a gap to flag in the paper.

H3 — Higher temperature decreases coherence

Pending   All runs in the current dataset use temperature 0.6. Temperature manipulation data (0.2, 0.6, 1.0) would be needed for this test. If earlier temperature-sweep runs are available in the raw data, we can add this analysis.

09 · Next Steps

Open Questions for Discussion

θ operationalisation: We recommend formally redefining θ as normalised extraction ratio in the Methods section, with a note explaining why the original pool-threshold definition is degenerate given the models' behaviour (immediate over-extraction from round 1).
Temperature sweep: If Gabriela has PD runs at multiple temperatures, and/or CD runs at different temperatures exist in the dataset, we can run H3 immediately. The pipeline already outputs by condition.
Factor analysis: With 6 models × 4 parameters, a formal latent factor analysis is underpowered (p < k). We can either (a) report the correlation matrix as evidence of structural independence, or (b) run a PCA for exploratory structure — both defensible for this N.
Opponent sensitivity (Δm): The game-level dataset includes condition labels (ai, human, human_prior, undisclosed). Δm = ||SmAI − SmHuman|| can be computed immediately from the existing data. Ready to run on request.
